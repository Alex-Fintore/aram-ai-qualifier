# AI-квалификатор входящих лидов

Клиентский сайт-бриф для подготовки пилота голосового помощника: воронка, сценарий звонка, интеграции, требования к данным, критерии успеха и бюджет.

Публичная версия: `https://alex-fintore.github.io/ai-qualifier/`.

## Возможности

- десять коротких вопросов на одной странице, из них пять обязательных;
- необязательный углублённый бриф только после отправки основных данных;
- локальное сохранение черновика в браузере;
- адаптивный интерфейс и доступная клавиатурная навигация;
- отправка через Formspree без секретов в публичном коде.

## Локальный запуск

```bash
npm install
npm run dev
npm test
npm run pages:build
```

## Канал ответов

Сборка читает публичный endpoint из `NEXT_PUBLIC_FORMSPREE_ENDPOINT`. В GitHub Actions он передаётся из одноимённой переменной репозитория. Пока переменная не задана, форму можно заполнить и скачать, но кнопка сетевой отправки честно отключена.

Не добавляйте в репозиторий исходный DOCX, расшифровку, клиентские данные, записи разговоров, `.env` или любые токены.
