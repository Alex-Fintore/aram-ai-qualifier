# AI-квалификатор входящих лидов

## Purpose

Клиентский сайт-бриф для первичной диагностики пилота голосового AI-квалификатора: текущая воронка, сценарий звонка, интеграции, безопасность, экономика и критерии успеха.

## Audience

- Руководитель продаж или операционного блока.
- IT/CRM и телефония для интеграционного раздела.
- Юрист или специалист по информационной безопасности для требований к данным.

## Delivery Shape

- Публичная адаптивная форма на GitHub Pages.
- Шесть смысловых разделов и финальная проверка.
- Черновик хранится только в браузере заполняющего.
- Получатель ответов подключается через внешний приёмник без секретов в публичном коде.

## Safety Invariants

- Не собирать персональные данные клиентов, записи разговоров, пароли или ключи доступа через публичную форму.
- Большие и чувствительные материалы запрашивать отдельно через согласованный безопасный канал.
- Исходный DOCX и расшифровка не входят в публичный репозиторий.
